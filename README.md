# ospfv3-dualstack-troubleshooting-lab
Fault-injected Cisco Modeling Labs (CML) topology for OSPFv3 + IPv4/IPv6 dual-stack troubleshooting. 6 IOSv routers, 3 OSPF areas, DHCPv4/v6, 3 workstations, and 10 embedded misconfigurations. Includes lab guide, solutions appendix, topology diagram, and portable YAML import file.
